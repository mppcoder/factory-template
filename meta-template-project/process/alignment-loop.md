# Контур выравнивания

## Когда использовать
Если defect найден в working project и может быть reusable для фабрики.

## Шаги
1. Создать bug report
2. Классифицировать defect layer
3. Если defect reusable — создать factory feedback
4. Проверить drift
5. Собрать patch export
6. Проверить safe apply
7. Обновить template repo
8. Обновить registry и meta-template-project
