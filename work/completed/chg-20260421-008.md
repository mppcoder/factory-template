# chg-20260421-008

## Заголовок
Flatten core-hot-15 reference export

## Что сделано
- `core_hot_direct` переведен на flat export layout через `packaging/sources/sources-profiles.yaml`.
- Export logic теперь кладет direct hot-set в одну папку без подпапок и пишет `source -> export_filename` в `manifest.json`.
- При потенциальном конфликте имён используется deterministic naming strategy `basename_unless_conflict_then_path_joined_with_double_underscore`.
- Docs и boundary instructions обновлены так, чтобы пользователь брал hot15 из одной flat-папки.

## Почему это важно
Ручное копирование и работа с reference/export артефактами больше не требуют навигации по вложенной структуре export-каталога.
