# chg-20260421-006

## Заголовок
Require immediate completion package in final answer

## Что закрыто
- completion package теперь должен быть в том же финальном ответе
- deferred instruction after user reminder теперь считается process defect
- done checklist и validator отражают immediate same-response rule

## Ключевые ограничения
- правило относится только к случаям, где completion package действительно обязателен
- release semantics не меняются
