# Техническая спецификация

## Изменяемые артефакты
- `src/ui/notify_form.html`
- при необходимости `docs/ui-copy.md`

## Технический подход
1. Найти строку с подписью кнопки в шаблоне.
2. Заменить `Сохранть` на `Сохранить`.
3. Проверить форму вручную в локальной среде.
4. Зафиксировать результат в verification-report.
