# Нормализованный контекст

## Проблема
При временных ошибках внешнего API уведомление помечается как неуспешное, но повторная попытка не выполняется автоматически.

## Что хотим получить
- retry только для временных ошибок;
- ограничение количества повторов;
- запись всех попыток в журнал;
- отсутствие бесконтрольных дублей.
