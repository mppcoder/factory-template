# T-002

## Название
Подключить retry к HTTP-каналу

## Цель
Сделать ограниченный retry для временных ошибок внешнего API.

## Основание
`.chatgpt/tech-spec.md`

## Входной контекст
- `.chatgpt/reality-check.md`
- `.chatgpt/task-index.yaml`
- `project-knowledge/fixes-registry.md`

## Действия
1. Подключить счётчик повторов к HTTP-каналу.
2. Добавить повторную постановку в очередь.
3. Сохранить идемпотентный ключ.
4. Обновить логирование попыток.

## Критерии приемки
- retry происходит только для временных ошибок;
- число попыток ограничено;
- идемпотентный ключ сохраняется.

## Риски
Ошибка в логике может создать дубли или пропуски.
