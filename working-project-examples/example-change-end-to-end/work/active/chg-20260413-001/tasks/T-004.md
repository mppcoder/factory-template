# T-004

## Название
Обновить документы и закрыть change

## Цель
Зафиксировать решение в знаниях проекта и закрыть жизненный цикл change.

## Основание
`.chatgpt/done-report.md`

## Входной контекст
- verification report
- fixes registry
- known issues

## Действия
1. Обновить known issues.
2. Добавить запись в fixes registry.
3. Заполнить done report.
4. Подготовить перенос work item в completed.

## Критерии приемки
- документы обновлены;
- done report заполнен.

## Риски
Если пропустить этот шаг, решение не станет частью проектной памяти.

- T-003
