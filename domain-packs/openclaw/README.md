# Доменный пакет: OpenClaw

Этот пакет не является частью core фабрики.
Он нужен как reference-case для brownfield-проектов с CLI, runtime и инфраструктурной спецификой.

## Что здесь хранится
- brownfield-checklists
- runbook-profiles
- evidence-patterns

## Что важно
- универсальный brownfield core не зависит от OpenClaw;
- OpenClaw здесь используется только как пример доменной настройки;
- этот слой не должен менять структуру core.
