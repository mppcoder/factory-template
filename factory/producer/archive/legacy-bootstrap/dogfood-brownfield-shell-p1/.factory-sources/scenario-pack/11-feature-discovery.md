# Анализ новой функции

Перед формированием решения обязательно пройди:
1. reuse check;
2. reality check;
3. conflict detection.

Только потом уточняй spec и decomposition.

## Если feature вскрыла скрытый дефект
Если в ходе feature work найден скрытый defect, нельзя bury it inside feature-only task. Нужно немедленно создать bug report и только после этого продолжать remediation.

## Правило скрытого дефекта
Если в feature work вскрывается hidden defect, нельзя bury it inside feature-only task. Нужно немедленно создать bug report и продолжать работу через defect-flow.
