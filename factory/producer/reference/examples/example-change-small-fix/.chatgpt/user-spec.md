# Пользовательская спецификация

## Что нужно изменить
Исправить опечатку в подписи кнопки сохранения на форме уведомления.

## Для кого это важно
Для пользователей интерфейса и для команды поддержки, которая получает обращения по качеству UI-текста.

## Ожидаемый результат
На кнопке должен отображаться текст `Сохранить` вместо `Сохранть`.
