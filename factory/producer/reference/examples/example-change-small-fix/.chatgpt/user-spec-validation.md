# Проверка пользовательской спецификации

- Граница изменения определена: затронут только один шаблон интерфейса.
- Целевой результат измерим: текст кнопки должен отображаться корректно.
- Вне объема: изменения API, миграции данных и перестройка логики формы.
- Спецификация достаточна для локального small-fix.
