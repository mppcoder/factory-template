# Краткое описание проекта

## Название проекта
Сервис уведомлений для внутренних команд

## Назначение
Проект отправляет уведомления о новых заявках в несколько каналов и хранит историю доставки.

## Что показывает этот пример
Мы разбираем change, в котором нужно добавить повторную отправку уведомления при временной ошибке интеграции.
