# Настройка repo-first инструкции в ChatGPT Project

Не загружай сценарии в Project как source of truth.
Настрой именно поле `Instructions` в ChatGPT Project до первого рабочего запроса.
Вставь короткую инструкцию: сначала открыть GitHub repo проекта, затем `template-repo/scenario-pack/00-master-router.md`, затем идти по маршруту из него.
Проверь, что в Project не осталось старого текста про `Sources`, staging-first поток или хранение сценариев внутри Project.
