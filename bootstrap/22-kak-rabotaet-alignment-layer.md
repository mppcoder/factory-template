# Как работает alignment layer

## Что делает alignment layer
Выравнивает три контура:
- factory improvement
- greenfield
- brownfield

## Что одинаково
- bug report
- defect classification
- factory feedback при reusable issue
- ChatGPT handoff при необходимости
- Codex bug-capture block
- запрет silent fixes
- done только после defect capture
