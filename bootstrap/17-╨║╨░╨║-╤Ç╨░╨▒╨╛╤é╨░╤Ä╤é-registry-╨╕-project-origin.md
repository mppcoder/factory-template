# Как работают registry и project origin

## Зачем нужен registry
Registry фиксирует:
- какие версии фабрики существуют;
- какие проекты созданы из фабрики;
- какие релизы были выпущены.

## Что такое project-origin.md
Это паспорт происхождения конкретного рабочего проекта.

## Что туда входит
- название проекта;
- slug;
- режим старта;
- версия фабрики;
- preset, class и execution mode на момент создания.

## Как использовать
1. Создайте проект через launcher.
2. Проверьте `.chatgpt/project-origin.md`.
3. При необходимости зафиксируйте запись в `factory/producer/registry/projects-created.md`.
