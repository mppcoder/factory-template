# Как создать рабочий проект

1. Запусти `./launcher.sh`
2. Выбери режим и класс изменения
3. Создай или открой нужный ChatGPT Project
4. В поле `Instructions` вставь repo-first инструкцию: сначала GitHub repo проекта, затем `template-repo/scenario-pack/00-master-router.md`, затем строго маршрут из него
5. Убедись, что в Instructions нет старого текста про `Sources`, staging-first поток или хранение сценариев внутри Project
6. Только после этого начинай работу через `template-repo/scenario-pack/00-master-router.md`
