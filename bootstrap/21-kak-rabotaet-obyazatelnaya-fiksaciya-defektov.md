# Как работает обязательная фиксация дефектов

1. Найден defect.
2. Создайте bug report в `reports/bugs/`.
3. Заполните classification.md и укажите слой дефекта.
4. Если defect reusable, оформите factory feedback.
5. При необходимости подготовьте ChatGPT handoff bug template.
6. Соберите Codex task pack.
7. Проведите verification и только после этого закрывайте задачу.
