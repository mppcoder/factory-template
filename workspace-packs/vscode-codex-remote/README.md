# Рабочий пакет: VS Code + Codex + Remote

Этот пакет не входит в core фабрики.
Он нужен только как дополнительный рабочий профиль.

## Для чего нужен
- работа с repo в одном окне;
- доступ к терминалу и файлам удаленной среды;
- удобный запуск Codex и локальных скриптов.

## Что не делает
- не меняет логику factory core;
- не обязателен для greenfield или brownfield;
- не влияет на сценарии, presets и validation.
