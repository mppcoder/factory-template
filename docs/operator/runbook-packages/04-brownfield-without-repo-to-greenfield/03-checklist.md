# Чеклист: путь без repo

- [ ] Target root создан как `/projects/<project-slug>`.
- [ ] `_incoming/` находится внутри target root.
- [ ] Temporary/reconstructed/helper repo не является sibling в `/projects`.
- [ ] Intake evidence заполнен.
- [ ] Reverse engineering plan/summary/decision log заполнены.
- [ ] Canonical repo определен или реконструирован.
- [ ] With-repo audit/adoption cycle выполнен.
- [ ] Active profile после conversion: `greenfield-product`.
- [ ] Lifecycle после conversion: `greenfield-converted`.
- [ ] Transitional workspace archived/renamed/moved out of active path.
- [ ] Verification green или documented blocker.
