# Закрытие: greenfield-product

## Критерии завершения

- Active profile: `greenfield-product`.
- Active mode: `greenfield`.
- Project lifecycle dashboard показывает next action или done state.
- Internal Codex-eligible follow-up закрыт: docs, validators, verification report, done report, changelog при необходимости.
- External actions ledger содержит только реальные external/manual/runtime/downstream действия.

## Внешние действия

Пользователь нужен только для внешних approvals, secrets, GitHub UI blockers или real deploy/release decisions.
Если таких действий нет, closeout пишет: `Внешних действий не требуется.`
