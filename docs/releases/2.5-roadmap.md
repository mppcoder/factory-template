## Треки

### Track 2.5-A — Engineering hardening

Целевые outcomes:
- deterministic routing and predictable launch boundary behavior;
- stronger guardrails/validators around handoff, release-facing consistency and closeout;
- safer defaults for downstream template consumers.

### Track 2.5-B — Beginner-first productization

Целевые outcomes:
- понятный first-run path без "экспертных" предположений;
- UI-friendly onboarding и release-facing тексты для не-экспертного пользователя;
- снижение количества ручных уточнений при первом рабочем цикле.

MVP считается закрытым, когда одновременно выполнено:

### Full 2.5 (DoD-Full)

Full 2.5 считается закрытым, когда одновременно выполнено:
- закрыт `G25-GA` и завершен release closeout;
